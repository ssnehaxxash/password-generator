# Password Generator

A simple Java program that generates a secure random password using letters, numbers, and special characters.

## Features
- Random password generation
- Combination of uppercase, lowercase, numbers, and symbols
- Adjustable password length

## Technologies Used
- Java
- Random class

## How to Run
1. Compile the Java file
2. Run the program
3. A random password will be displayed in the console

## Learning Outcome
Practiced working with random number generation and string manipulation in Java.
